# Challenge_TelecomX
Challenge de Ciencia de Datos
# 📊 Análisis de Evasión de Clientes (Churn)

## 🔹 Introducción
Este proyecto analiza el fenómeno de **evasión de clientes (Churn)** en una empresa de telecomunicaciones.  
El objetivo es identificar patrones en los datos que permitan comprender por qué los clientes cancelan sus servicios y proponer estrategias de retención.

---

## 🔹 Requisitos y dependencias
El proyecto está desarrollado en **Python 3.9+** y utiliza las siguientes librerías:

- pandas
- numpy
- matplotlib
- seaborn
- jupyter

Instala las dependencias con:

Instalación
- Clona este repositorio:
git clone (https://github.com/JCoronelR/Challenge_TelecomX)
- Entra en la carpeta del proyecto:
cd Challenge_TelecomX
- Instala las dependencias:
pip install -r requirements.txt

🔹Ejecución del proyecto
- Inicia Jupyter Notebook:
jupyter notebook
- Abre el archivo:
churn_analysis.ipynb
- Ejecuta las celdas paso a paso para reproducir el análisis y visualizar los gráficos.

🔹 Posibles problemas y soluciones
- Error de versión de Python: Asegúrate de usar Python 3.9 o superior.
- Dependencias faltantes: Ejecuta nuevamente pip install -r requirements.txt.
- Gráficos no se muestran: Verifica que estás ejecutando el notebook en un entorno Jupyter, no en un script plano.
- Encoding de datos: Si el dataset tiene caracteres especiales, usa encoding='utf-8' al cargarlo.

🔹 Resultados esperados
El notebook incluye:
- Limpieza y tratamiento de datos.
- Análisis exploratorio con gráficos (histogramas, boxplots, barras).
- Conclusiones sobre los factores que influyen en la evasión.
- Recomendaciones estratégicas para reducir el churn.
